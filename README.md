# dotfiles
My dotfiles

## Installation
* Install OMF
  ```fish
  curl -L https://get.oh-my.fish | fish
  ```
* Update fisher and install plugins
  ```fish
  fisher self-update
  ```

  ```fish
  fisher
  ```
* Install vim plugins 
  ```vim
  :PlugInstall
  ```
* Install fonts
  ```fish
  brew tap homebrew/cask-fonts
  brew cask install font-hack-nerd-font
  ```
